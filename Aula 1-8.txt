[Nós criamos um componente de texto que atendia parcialmente o que precisamos, modificamos e evoluímos.
Agora está na hora de encaixarmos os componentes de texto nesse formulário.]

Crie uma pasta Formulario dentro da pasta componentes, que terá os arquivos index.js e
Formulario.css. No index.js, faça o import do Formulario.css, crie uma arrow function com
export. Dentro do return, crie as tags <section> e <form>, uma dentro da outra, e ponha as
tags do CampoTexto dentro.
Como ele não sabe o que é a tag CampoTexto, você precisa importá-la da pasta CampoTexto OU
posicionar o mouse sobre a tag e apertar CTRL. O VSCode vai dar uma sugestão. Clicando nela
o import é feito automaticamente.
Assim:


```
import CampoTexto from "../CampoTexto";
import './Formulario.css';

const Formulario = () => {
    return (
        <section>
            <form>
                <CampoTexto label="Nome" placeholde="Digite seu nome"/>
                <CampoTexto label="Cargo" placeholder="Digite seu cargo"/>
                <CampoTexto label="Imagem" placeholder="Digite o endereço da imagem"/>
            </form>
        </section>
    )
}

export default Formulario
```

Agora devemos ir ao App.js, colocar o <Formulario /> debaixo do banner e importar.
Depois voltamos ao index.js do Formulario e colocamos um título h2 abaixo do <form>.
Crie uma classe para a section e faça as devidas propriedades para o formulario, o form
e (opcional) o h2.
