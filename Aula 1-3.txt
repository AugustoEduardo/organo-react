Primeiro, certifique-se de ter o node.js instalado. Basta entrar no prompt ou no PowerShell e digitar:
node -v
ou
node --version

Daí, com o prompt / PowerShell aberto, execute o comando npx create-react-app organo

(organo é o nome do app que vamos criar, o nome do projeto)

Conforme ele for criando o projeto, o programa vai puxar alguns pacotes para nos ajudar a trabalhar
com React, está preparando diretórios em nossos arquivos para poder separar onde fica o CSS e onde fica
o Javascript. Esta preparação é o que chamamos de *template*.


Quando inicializar o React, dentro da pasta organo, vai haver um arquivo .gitignore.

Abra a pasta organo (no PowerShell: cd organo) e rode o comando npm start.

["Agora não usamos o npx mas sim o npm, a diferença é que o npm nós vamos utilizar para
rodar coisas locais, quando rodamos o npx quer dizer que estamos rodando o script de forma remota,
não está muito vinculado ao ambiente local."]



[You can now view organo in the browser.

  Local:            http://localhost:3000
  On Your Network:  http://192.168.15.72:3000]


Note that the development build is not optimized.
To create a production build, use npm run build.

webpack compiled successfully




Não fique assustado no início. O React tem estilos de código que misturam HTML com Javascript,
por isso temos uma function e algumas tags ao inicializar o App.js.

["Em vez de criarmos os elementos e irmos fazendo document.querySelector, document.createElement
para manipularmos o DOM, vamos utilizar o React e uma dessas formas de fazermos isso é
criar componentes. Então, o que vocês veem aqui em function App() nada mais é do que uma forma
de escrever um componente React."]
