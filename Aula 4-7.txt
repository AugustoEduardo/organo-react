Como a gente faz para exibir somente as categorias com pelo menos 1 colaborador?
E como deixar os campos do formulário vazios após criar um card?


OBS: Esta parte pode ser bastante difícil de entender para quem está muito acostumado
com a linguagem Javascript, então preste atenção!


O jeito de ocultar as categorias é ir no index.js do Time e fazer um "condicional sem if".
Não podemos usar "if (algumasCondições) {}" porque a linguagem do React é JSX e esta possui
algumas peculiaridades que a difere do Javascript.
Então, repare que tudo o que está na <section> é a exibição das equipes. Antes dela,
escreveremos a condição de que o número de colaboradores precise ser maior que zero.
E - atenção aqui - a forma de verificar uma condição é através do "&&".


props.colaboradores.length > 0 && <section className="time" style={cssEstilo}>
.......
</section>


O código acima lê-se: "Se a quantidade de colaboradores for maior que zero, execute
o que estiver à direita (no caso, a section responsável pela criação dos times)".
Diferente do Javascript onde o "&&" era usado para combinar duas condições, no React
ele é usado para "se a condição da esquerda é satisfeita, execute o que tem à direita".



Se preferir, tem uma maneira um pouco mais fácil de entender.
Usamos o "?" em conjunto com ":" para mostrar que devemos executar algo caso a primeira condição
seja verdadeira e executa outra coisa caso seja falsa.


props.colaboradores.length > 0 ? <section className="time" style={cssEstilo}>
<h3 style={{borderColor: props.corPrimaria}}>{props.nome}</h3>
    <div className="colaboradores">
    {props.colaboradores.map (colaborador => <Colaborador
            nome={colaborador.nome}
            cargo={colaborador.cargo}
            imagem={colaborador.imagem}
        />)}.
        </div>

    </section>
    : ""


"Se o tamanho dos colaboradores for maior que zero, executo a section. Caso contrário,
retorno vazio".



Ao recarregar a página, verá que todos os times sem colaboradores já não aparecem mais.
Por fim, vamos resetar os campos do formulário quando clicamos no botão Criar Card.

É bem simples: vá ao index do Formulario, na função "aoSalvar", peça para setar os
quatro campos para vazio.


const aoSalvar = (evento) => {
    evento.preventDefault()
    props.aoColaboradorCadastrado({
        nome,
        cargo,
        imagem,
        time
    })

    setNome('')
    setCargo('')
    setImagem('')
    setTime('')

}
